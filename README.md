## Interactive Ideal Gas Simulator

# Instructions

| Key | Description |
| --- | --- |
| `W` | Forward movement |
| `A` | Rightward movement |
| `S` | Backward movement |
| `D` | Leftward movement |
| `E` | Upward movement |
| `Q` | Downward movement |
| `UP/DOWN` | Upward / downward rotation |
| `LEFT/RIGHT` | Leftward / Rightward rotation |
